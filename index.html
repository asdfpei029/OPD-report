判讀邏輯 Algorithm

1. 肺功能 PFT / Flow Volume
  • Obstructive：Pre 或 Post FEV1/FVC < 70%
  • Suspected Obstructive：非Obs 且 FEF25-75 %pred < 65%
  • Suspected Restrictive：非上述 且 FVC %pred < 80%
  • Normal：以上皆非
2. BDT 支氣管擴張試驗
  • Positive：Post FEV1 或 FVC ↑ >12% 且 >200ml
  • Partial Positive：↑ >12% 或 >200ml（符合其一）
3. IOS 震盪肺功能

Step 1 Screening（任一符合）

  • R5–R20 > 0.07
  • X5 < −0.12
  • Fres > 14.14
  • AX > 0.44

Step 2 Classification (ΔX5 = X5pred − X5meas)

  • Normal：R5 <140% & ΔX5 <0.15
  • Restriction：R5 <140% & ΔX5 ≥0.15
  • Central Obstruction：R5 ≥140% & ΔX5 <0.15
  • Peripheral Obstruction：R5 ≥140% & ΔX5 ≥0.15 & R5–R20 ≥0.07
  • Probably Central+Restriction：R5 ≥140% & ΔX5 ≥0.15 & R5–R20 <0.07
4. CT / LDCT
  • 結節：Total number、各 #N 詳細、Nodules else 肺葉、Atypical cyst、Perifissural
  • 智慧合併：概括項目（<6mm / benign）與 #N 詳細用 IM 號碼比對後合併
  • Comparison：有舊片時抓日期；No prior CT 自動標注
  • Other Findings：Emphysema、Bronchiectasis、Coronary Ca、LN 等
  • 順序：Nodules → Lung-RADS → No prior CT → Other
  • 過濾：Quality 描述、醫師簽名
5. Gemini Vision 模式
  • 貼圖後直接送 Gemini 2.5 Flash-Lite API
  • 自動辨識 PFT 表格並套用判讀邏輯
  • 支援任何廠牌機台、任何欄位順序
  • 需要在上方填入 Gemini API Key
